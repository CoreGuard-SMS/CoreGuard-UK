# Logo Instructions

This directory contains the logos for CoreGuard SMS.

## Current Files:

1. **company-logo-light.png** - Light theme version of the CoreGuard SMS logo
2. **company-logo-dark.png** - Dark theme version of the CoreGuard SMS logo
3. **company-logo.png** - Backup copy (light theme)
4. **employee-logo.png** - Backup copy (dark theme)

## Logo Usage:

### Sidebar Logo
- **Light Theme**: Uses `company-logo-light.png`
- **Dark Theme**: Uses `company-logo-dark.png`
- **Fallback**: Shield icon if logos fail to load

### Landing Page Header
- **Light Theme**: Uses `company-logo-light.png` (h-12 size)
- **Dark Theme**: Uses `company-logo-dark.png` (h-12 size)

### Landing Page Features
- **Security Section**: Uses theme-appropriate logo (h-5 size)
- **Compliance Badge**: Uses theme-appropriate logo (w-12 h-12 size)

## Logo Specifications:

- **Format**: PNG with transparency
- **Original Size**: Downloaded from external URLs
- **Scaling**: Automatically scaled to fit containers
- **Theme Support**: Separate versions for light/dark themes
- **Style**: Professional CoreGuard SMS branding

## Fallback:

If the logo files fail to load, the system will automatically fall back to the shield icon.

## Updating Logos:

1. Replace the logo files in this directory
2. Keep the same filenames for compatibility
3. Restart your development server to see the changes
4. Ensure both light and dark versions are available for best results
