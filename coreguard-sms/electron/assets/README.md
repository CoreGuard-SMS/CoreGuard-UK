# Assets Directory

This directory contains static assets for the CoreGuard Control desktop application.

## Required Files

### icon.ico
- **Purpose**: Application icon for Windows
- **Size**: 256x256 pixels recommended
- **Format**: ICO format with multiple sizes (16x16, 32x32, 48x48, 256x256)
- **Location**: Place in this directory as `icon.ico`

## Adding the Icon

1. Create or obtain a high-quality CoreGuard logo
2. Convert to ICO format with multiple sizes
3. Save as `icon.ico` in this directory
4. The build process will automatically include it

## Icon Requirements

- **Square format**: Equal width and height
- **Transparent background**: Recommended for professional look
- **High resolution**: 256x256 for best quality
- **Brand consistent**: Use CoreGuard branding guidelines

## Tools for Icon Creation

- **Online**: favicon.io, convertico.com
- **Desktop**: GIMP, Photoshop, IcoFX
- **Command line**: imagemagick, png2ico

## Current Status

⚠️ **Action Required**: Add the `icon.ico` file to complete the application setup.

The application will build without the icon, but will use the default Electron icon.
